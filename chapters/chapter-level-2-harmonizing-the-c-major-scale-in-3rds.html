Level 2 · Chapter 3

Harmonizing the C Major Scale in 3rds

Chapter objective. Derive the 3rd above every note of C Major and explain why the resulting 3rds are not all the same quality.

Retrieve first. Write C Major from memory. Above C and then above D, build a 3rd and label each Major or minor before consulting the text.

Review point. Use this written pattern again in Chapter 4, then turn it into triads in Chapters 5–9.

First, here is the C Major scale written out horizontally with each note's scale degree underneath.

Notes ➡ C D E F G A B C
Scale Degrees ➡ 1̂ 2̂ 3̂ 4̂ 5̂ 6̂ 7̂ 8̂ or 1̂

The 1-3 Formula

The "formula" is 1-3. Assign any note to be "1". Then count up to 3 to get the next note.

If C is 1, then E is 3 because C (1), D (2), E (3). This first pair is very easy to see because C is the 1st scale degree and E is the 3rd scale degree.

3 ➡ E
1 ➡ C D E F G A B C

Let's continue. If D is 1, then F is 3 because D (1), E (2), F (3).

3 ➡ E F
1 ➡ C D E F G A B C

And the rest:

3 ➡ E F G A B C D E
1 ➡ C D E F G A B C

Some 3rds are Major, some are minor

Every pair is a 3rd, but they are not all the same kind of 3rd. Count the half steps to find the type of 3rd:

  • C–E: 4 half steps
  • D–F: 3 half steps
  • E–G: 3 half steps
  • F–A: 4 half steps
  • G–B: 4 half steps
  • A–C: 3 half steps
  • B–D: 3 half steps

If a 3rd spans 4 half steps, it is a Major 3rd. If it spans just 3 half steps, it is a minor 3rd.

3 ➡ E F G A B C D E
1 ➡ C D E F G A B C
M3 = Major 3rd,
m3 = minor 3rd ➡		 M3 m3 m3 M3 M3 m3 m3 M3

Why this matters

"Stacked" 3rds are the raw material of harmony and chords. A 3rd is the building block of a chord. In Chapter 5, we'll get into that more.

Try it yourself

Cover the completed tables. Write each scale from tonic to tonic, add the note a 3rd above every scale note, and label every pair Major or minor.

  1. Harmonize C Major in 3rds.
  2. Transfer the same process to G Major. Remember that G Major contains F♯.
Answer key and success standard

C Major: C-E Major, D-F minor, E-G minor, F-A Major, G-B Major, A-C minor, B-D minor, C-E Major.

G Major: G-B Major, A-C minor, B-D minor, C-E Major, D-F♯ Major, E-G minor, F♯-A minor, G-B Major.

Pass when both rows have all eight upper notes and qualities correct without reopening the tables. If one pair is wrong, rebuild that pair by counting note names and half steps; do not recopy the whole answer row.
