Level 2 · Chapter 31

The Fretboard: Sus Chords on Barre Shapes

Chapter objective. Locate the movable 2, minor 3rd, Major 3rd, and 4 around a barre-chord root, then use a sus4-to-Major resolution deliberately inside a progression.

Retrieve first. Build C Major and C minor as A-shape barres. Name the B-string note that is the 3rd, then predict where C's 2nd and 4th lie.

Assessment point. Complete the Unit 6 cumulative assessment at the end of this chapter. It checks Roman-numeral understanding, chord-function language, movable shapes, and a musically placed sus resolution.

The theory was small: a sus chord replaces the 3rd. Now your hand gets to feel why that matters.

In both barre shapes, the 3rd is one finger. That means sus2, minor, Major, and sus4 can line up on one string like four stops on the same road.

One finger tells the whole story.

The C barre line: 2, minor, Major, 4

Use the A-shape C barre chord at the 3rd fret. The root is C on the A string. The 5th is G. The mood note sits on the B string.

On that B string, the four useful notes sit in four consecutive frets:

B-string fretDegree over CChord
32Csus2
4lowered 3rdC minor
53C Major
64Csus4

So the line is:

Csus2 → C minor → C Major → Csus4

Do not rush past that. The same root and 5th keep ringing while one finger walks up the B string. Open, dark, bright, tense.

How to hold it

Plant the A-shape C barre first. Keep the A string root and the D/G string power-chord frame steady. Then move only the note on the B string.

  • Csus2: B string, 3rd fret.
  • C minor: B string, 4th fret.
  • C Major: B string, 5th fret.
  • Csus4: B string, 6th fret.

You may need to simplify the grip at first. That is fine. If you can keep C and G ringing while the B-string note moves, your ear will hear the whole lesson.

The E-shape version

The E-shape barre gives you the same idea on a different string. Use G at the 3rd fret. In that shape, the mood note sits on the G string.

Here is the one-finger line:

G-string fretDegree over GChord
22Gsus2
3lowered 3rdG minor
43G Major
54Gsus4

Different chord shape, same principle. The root and 5th make the frame. One note inside the frame decides the color.

Use sus4 before the dominant resolves

In the last two chapters you played I–IV–V progressions. The V chord already wants to fall home to I. Sus4 makes that pull even stronger because it lets the 4 fall to the 3 before the whole chord resolves.

In the key of C, try this:

C → F → Gsus4 → G → C

That small Gsus4-to-G move is one of the most familiar guitar sounds. The suspended note relaxes first, then the dominant goes home.

Hear it

Play the C barre line slowly, one strum per chord:

Csus2 → C minor → C Major → Csus4 → C Major

Listen for the four personalities. Sus2 feels open. Minor darkens. Major brightens. Sus4 leans forward and wants to fall back to Major.

Try it yourself

Build the same four-stop line for D using the A-shape barre. D lives at the 5th fret of the A string, so the moving note is on the B string.

  1. Which B-string fret gives you D Major?
  2. Which fret gives you Dsus4?
  3. Which fret gives you Dsus2?
Answer key
  • D Major: B string, 7th fret. That is the 3rd of D.
  • Dsus4: B string, 8th fret. Move the 3rd up one fret to the 4th.
  • Dsus2: B string, 5th fret. Move two frets below the Major 3rd to the 2nd.

The takeaway is practical: if you can find the 3rd in a barre chord, you can find the sus chords beside it. The theory is 1-2-5 and 1-4-5; the fretboard version is one finger moving around the mood note.

Unit 6 cumulative assessment: pattern, function, and resolution

Do this with the book closed until you need to check a result. It combines the entire function-and-progression unit.

  1. In G Major, write I-IV-V-I as chord symbols. Label each chord's broad function: home, away, pull, home.
  2. Choose an E-shape/A-shape realization that keeps the progression in one practical region. Write each root string and fret, then play two four-beat measures per chord at 60 bpm.
  3. On the V chord, add a sus4 for two beats, resolve it to the Major V for two beats, then return to I. Name the changed chord degree and the direction of its resolution.
  4. Without a diagram, transfer the I-IV-V pattern to A Major and name the three chords.
Pass rule and spacing

Pass when the numeral-to-chord translation and root locations are correct, the pulse survives the shape changes, and the sus4 resolves to the 3rd before the dominant returns home. At the next practice session, repeat only the task that was weakest. After three further sessions, run the whole assessment again without preparation; that delayed attempt is the meaningful score.
