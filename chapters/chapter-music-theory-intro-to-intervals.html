Chapter 11

Music Theory: Intro to Intervals

An interval is the measurement of distance between any two named notes. They are calculated by counting the lowest note as 1. Before calculating intervals, I’m going to first re-arrange the C Major Scale graphic to look more like a piano.

One heads-up before we start counting: back in the Major Scale chapter, the numbers in “221-2221” counted frets (half steps). From here on, naming an interval—a “2nd,” a “5th,” and so on—means counting letter names instead. It’s the same word doing two jobs; we’ll return to half-step counting in the next couple of chapters.

The 2nd Interval

A “2nd” interval is an interval that spans 2 note names. For example, C–D is a 2nd interval. C (1) and D (2). Count just the named notes in the circle:

C–D is an example of a whole step. Notice that you skip over the unnamed black note in between C and D. E–F is also a 2nd because it spans two names.

E–F is an example of a half step. In summary, to figure out an interval just start counting from 1 and only count the letters. Counting intervals is a foundational for when we get to creating power chords (and other types of chords).

The 3rd Interval

A 3rd is an interval that spans 3 note names. For example, D–F is a 3rd interval because 3 notes in the scale are spanned: D (1), E (2), and F (3).

G–B

G–B is a 3rd because it spans 3 notes in the scale: G (1), A (2), and B (3).

B–D is a 3rd

To count from B to D, wrap around to the beginning of the scale after you have reached C but don’t count C twice. B (1), C (2), D (3).

The 4th Interval

A 4th is an interval that spans 4 note names. For example, E–A is a 4th because it spans 4 notes in the scale: E (1), F (2), G (3), and A (4).

You count all intervals in the same manner—5ths, 6ths, 7ths, and 8ths. Start counting from 1 and just count the letter names until you have reached the higher note.

The 1st and 8th intervals are also called the unison and the octave respectively.

The unison interval “spans” only 1 note name. In other words, it’s the same note. On the guitar, you can utilize the unison interval to help you tune your guitar. For example, when you play the 5th fret of the E string, it produces the note A. This note A should be in unison with the open A string. In this case, A–A is a unison interval because you only span 1 note name.
